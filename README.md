# FocusGate Web

Admin Dashboard for FocusGate Gateway.
Built with Next.js 16, Tailwind CSS, and MongoDB.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Setup environment variables in `.env.local`
4. Run development server: `npm run dev --webpack`

## Features
- Real-time SMS monitoring
- SIM Card status tracking
- Wallet balance management
- Admin user controls
